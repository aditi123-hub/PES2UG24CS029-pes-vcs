## Project Overview
PES-VCS is a mini version control system implemented in C. It supports repository initialization, object storage, staging, commits, logs, and references similar to Git.

## Features
- Initialize repository
- Store objects using SHA-256
- Stage files using index
- Create commits
- View commit history
- Maintain HEAD and branch refs

## Phase 1
Implemented object storage, hashing, deduplication, and object tests.

## Phase 2
Implemented tree parsing, tree serialization, and tree tests.

## Phase 3
Implemented index staging area, add command, and status command.

## Phase 4
Implemented commits, log history, refs, and HEAD updates.
